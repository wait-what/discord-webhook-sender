How to get a webhook URL

You need to have the "Manage webhooks" permission (or be an admin) to create a webhook. Go to your server's settings > Webhooks > Create webhook > Set the name, channel and avatar. > Copy the URL > Save the webhook > Done.
You can later rename and delete it through this program or in the server's settings.
